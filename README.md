# Battleship Game

![Live demo](live-demo.png)

## [Live Demo](https://tarikkocar.github.io/battleship/)

Battleship game from The Odin Project curriculum. Created using test-driven development (TDD) principles.

Built with:

- HTML
- CSS
- JavaScript
- Webpack
- Jest
